Ma'lumotlarni arrayga saqlasangiz bo'ladi mock data qilib

# 1-topshiriq

	## Databaseda ma'lumot ko'rinishi

	meals
	-id
	-name
	-price
	-quantity
	-category_id

	categories
	-id
	-name

	## GraphQL schema

	type Query {
		meals: [Meal!]!
		categories: [Category!]!
	}

	type Meal {
		id: ID!
		name: String
		price: Float
		quantity: Int
		category: Category
	}

	type Category {
		id: ID!
		name: String
		meals: [Meal!]
	}

	## Tavsif
	Yuqoridagi ma'lumotni db (array)da saqlanish strukturasi, graphql schemadan foydalanib graphql server quring

	## GraphQL schema

	}

	type Post {
		id: ID!

		content: Float
		user: User
		comments: [Comment!]!
	}

	type User {
		id: ID!
		name: String
		posts: [Post!]!
	}

	type Comment {
		id: ID!
		text: String

	}
